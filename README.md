# ASCII Art Generator

A CLI tool that generates beautiful ASCII art text with colorful gradients, inspired by the Claude Code terminal logo.

## Features

- Generate giant ASCII art from any text
- Colorful gradient effects
- Filled block character style
- Easy-to-use CLI interface

## Installation

```bash
npm install
```
